Upload Thiết bị hàng loạt bằng Excel

Hướng dẫn định dạng file Excel:
  • File phải là `.xlsx` hoặc `.xls`.
  • Dòng đầu tiên phải là tiêu đề của cột.
  • Các cột sau là bắt buộc:
    • name (Tên thiết bị)
    • serial (Số Serial, phải là duy nhất)
  • Các cột sau là tùy chọn (nếu thiếu sẽ dùng giá trị mặc định):
    • category (Danh mục, vd: 'Linh kiện', 'Công cụ')
    • description (Mô tả)
    • location (Vị trí, vd: 'Kho A', 'Tủ 1')
    • unit (Đơn vị tính, vd: 'Cái', 'Bộ', 'Mét'. Mặc định: 'Cái')
